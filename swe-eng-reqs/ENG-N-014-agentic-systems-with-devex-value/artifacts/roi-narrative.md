# Fixture ROI Narrative

The demo compares one deterministic failed-build baseline with an offline
agent-assisted diagnose, retrieve, and propose path. It reports scenario-level
minutes saved and MTTR improvement to make the value hypothesis inspectable.

These values are synthetic fixtures, not production telemetry. The API always
returns `baseline_source=fixture` and `fabricated_prod=false`; real adoption,
quality, and incident measurements would be required before making a production
ROI claim.
