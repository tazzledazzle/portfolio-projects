Update CLAUDE.md Environment table with `OTEL_EXPORTER_OTLP_ENDPOINT` and remove/replace "No observability stack" under Known Deliberate Omissions. For compose (D-07): add an optional Compose profile `observability` (Prometheus/Loki/Tempo/Grafana/Alloy) so default `docker compose up -d` stays unchanged.
